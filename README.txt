# Recursos del Experimento de Robot Balancín

En esta carpeta se encuentran los recursos utilizados para el desarrollo 
del experimento del robot balancín. El proyecto fue realizado por:

- Fabián Miranda
- Tomás Araya
- Tomás Montaner

### Contenido de la carpeta:
1. **Archivos STL**: Modelos 3D para las piezas del robot.
2. **Código Arduino**: Código fuente para controlar el robot.
3. **Informe del proyecto**: Documento con la descripción, metodología, resultados y conclusiones del proyecto.

¡Gracias por revisar nuestro trabajo!
